# Цель работы

Приобретение практических навыков взаимодействия пользователя с системой посредством командной строки.

# Ход работы

1. Определяем полное имя домашнего каталога. Далее переходим в каталог /tmp, выводим содержимое этого католога с разными параметрами.
2. Определяем, есть ли в каталоге /var/spool подкаталог с именем cron
3. Переходим в домашний каталог, затем выводим все его содержимое.
4. Создаем каталог newdir и подкаталог morefun
5. Далее создаем одной командой три каталога - letters, memos, misk. После все эти три каталога удаляем.
6. Удаляем каталог newdir вместе с подкаталогом morefun
7. Узнаем набор опций команды ls для просмотра содержимого каталогов.
8. Узнаем опции для сортировки по времени и выдачи полной информации о файлах.
9. Далее получем ифнормацию об опциях команд ```cd```, ```pwd```, ```mkdir```, ```rmdir```, ```rm```
10. Узнаем историю выполненых команд, и вызываем команды с внесением изменений



# Вывод
Мы преобрели навыки взаимодействия с системой посредством использования командной строки.
